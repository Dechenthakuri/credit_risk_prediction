# Credit Risk Prediction 🏦

This project uses public credit data to explore and analyze default risk using Python, Pandas, NumPy, and data visualization tools.

## 🧰 Tools Used
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Jupyter Notebook

## 📊 Key Steps
- Data cleaning and handling missing values
- Exploratory data analysis (EDA)
- Feature engineering
- Visualizations
- Optional: basic machine learning (Logistic Regression)

## 📈 Insights
- Default rate is higher among low-income individuals.
- Debt ratio correlates with default likelihood.

## 🚀 How to Run
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the notebook: `jupyter notebook credit_risk_analysis.ipynb`
